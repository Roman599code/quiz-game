# Quiz Game (Vanilla JavaScript)

A simple and responsive Quiz Application built as part of my 100 Days of Code challenge.

## Features
- Dynamic question rendering from a JSON-like array.
- Live score tracking and progress bar animation.
- Instant visual feedback (green for correct answers, red for incorrect).
- Interactive final results screen based on user performance.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (DOM Manipulation, Array Methods, Dataset API)
